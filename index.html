Introduction to PHP 8

What is PHP?

PHP (Hypertext Preprocessor) is a widely-used open-source scripting language that is particularly suited for web development.

It is embedded within HTML code and executed on the server, generating dynamic content that can be sent to the client's web browser.

Using PHP in HTML

To embed PHP code in an HTML file, use the opening <?php tag and the closing ?> tag. Everything between these tags is treated as PHP code. 

<?php
    // Your PHP code here
?>

Commenting Code

Single line comments can be written by prefixing with //

<?php
    // Single line comment
?>

As an alternative to the double slash, PHP also supports the Perl style comments. Where # is used as the character to indicate that the rest of the line will be treated as a comment.

Multi lines comment can be written using /* comment */ syntax. 

<?php
    # Title: My first PHP script
    /*
        Multi
        line
        comment
    */
?>

Outputting Data to the Client

PHP provides several methods to output data to the client's browser. The most common are:

  • echo() statement
  • printf() function
  • sprintf() function
  • print_r() function
  • var_dump() function

Declaring Variables and Constants

In PHP, variables are declared using the dollar sign ($) followed by the variable name. PHP is loosely typed, meaning you don't need to declare the data type explicitly. 

<?php
    $name = "John";
    $age = 25;
?>

Constants are declared using the define() function. Once set, a constant's value cannot be changed during the script's execution. 

<?php
    define("PI", 3.14);
    define("SITE_NAME", "My Website");
?>

String Interpolation

String interpolation allows you to embed variables directly into a string. This is done using double quotes ("") or using the curly braces ({}) syntax. 

<?php
    $name = "John";
    echo "My name is $name.";
    // or
    echo "My name is {$name}.";
?>

Data types and operators

PHP Data Types

PHP supports several data types:

  • String: Textual data enclosed in quotes.
  • Integer: Whole numbers without decimal points.
  • Float (Double): Numbers with decimal points.
  • Boolean: Represents true or false.
  • Array: An ordered, indexed collection of values.
  • Object: Instances of user-defined classes.
  • NULL: Represents a variable with no value.

Operators in PHP

Operators perform operations on variables and values. PHP supports various operators, including arithmetic, assignment, comparison, logical, and more. 

<?php
    $a = 5;
    $b = 3;

    $sum = $a + $b;      // Addition
    $difference = $a - $b; // Subtraction
    $product = $a * $b;   // Multiplication
    $quotient = $a / $b;  // Division
    $remainder = $a % $b; // Modulus
?>

Expressions in PHP

PHP supports various expressions, which are combinations of values and operators that result in a single value. For example: 

  <?php
      $sum = 10 + 5;      // Addition
      $difference = 10 - 5; // Subtraction
      $product = 10 * 5;   // Multiplication
      $quotient = 10 / 5;  // Division
      $remainder = 10 % 3; // Modulus
  ?>

Control structures in PHP

Different types of control structures in PHP

PHP supports Various types of control structures

  • If Statement
  • Switch Statement
  • For loop
  • While loop
  • Break and Continue Statements

Conditional Statements

If Statement

<?php
    $x = 10;

    if ($x > 5) {
        echo "The variable is greater than 5.";
    } elseif ($x == 5) {
        echo "The variable is equal to 5.";
    } else {
        echo "The variable is less than 5.";
    }
?>

Switch Statement

<?php
  $day = "Monday";

  switch ($day) {
      case "Monday":
          echo "It's the beginning of the week.";
          break;
      case "Friday":
          echo "It's almost the weekend!";
          break;
      default:
          echo "It's a regular day.";
  }
?>

Looping Statements

For Loop

<?php
  for ($i = 0; $i < 5; $i++) {
    echo $i;
  }
?>

While Loop

<?php
  $counter = 0;

  while ($counter < 5) {
      echo $counter;
      $counter++;
  }
?>

Break and Continue Statements

<?php
  for ($i = 0; $i < 10; $i++) {
    if ($i == 5) {
        break; // Exit the loop when i equals 5
    }
    if ($i % 2 == 0) {
        continue; // Skip even numbers
    }
    echo $i;
  }
?>

Functions in PHP

Types of functions

Functions are blocks of reusable code that can be defined and executed in a program. They allow you to encapsulate a set of actions, making your code more modular and easier to maintain. PHP supports various types of functions:

  • Built-in Functions: echo(), strlen() etc.
  • User-Defined Functions
  • Anonymous Functions (Closures)
  • Recursive Functions
  • Variadic Functions

Declaring Functions

A function definition looks like this:

<?php
  function sayHello() {
    echo "Hello, World!";
  }
?>

Function Parameters and Arguments

Parameters are variables used in a function declaration to represent the values that a function expects to receive when it is called. They act as placeholders for the actual values (arguments) that will be passed to the function during its invocation.

Arguments are the actual values that are passed to a function when it is called. 

<?php
  //Function with parameter
  function greet($name) {
    echo "Hello, $name!";
  }

  //Calling function with argument
  greet("John");
